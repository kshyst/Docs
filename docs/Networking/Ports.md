# Ports

Any detail about web protocol ports goes here

| Port | Usage |
|-----|-------|
| 20, 21 | FTP (one for data, one for control) |
| 22 | SSH |
| 23 | Telnet |
| 25 | SMTP |
| 53 | DNS |
| 80 | HTTP |
| 110 | POP3 |
| 123 | NTP |
| 139 | NetBIOS |
| 143 | IMAP |
| 161, 162 | SNMP |
| 389 | LDAP |
| 443 | HTTPS |
| 465 | SMTPS |
| 636 | LDAPS |
| 993 | IMAPS |
| 995 | POP3S |


> Note that in this table all ports above 400 ends with S, which stands for Secure

You can find all of the above ports and many many others in `/etc/services`
